BPM‑Buddy (Android) — Laufmusik im Takt

Live-Cadence → Auto‑Playlist → Crossfade (optional Time‑Stretch).

MVP: Smartphone‑Beschleunigung erfasst Schritte in Echtzeit, wählt passende Tracks (½×/1×/2×‑Mapping) und fadet taktbewusst. Optional on‑device KI (PyTorch Lite oder TFLite) für robuste Schritt‑Erkennung.

Features

Echtzeit Cadence (Schritte/min) aus dem Smartphone‑Accelerometer

Auto‑Trackwahl nach BPM (80–180 BPM, inkl. ½×/2×‑Mapping)

Crossfade zwischen zwei Playern (Media3/ExoPlayer)

Modularer Step‑Detector: DSP (Baseline) oder ML (TFLite / PyTorch Lite)

Tracks aus Assets (assets/media) + Metadaten in tracks.json
